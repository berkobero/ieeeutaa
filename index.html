Berkobot

Berkobot is a multi-purpose discord bot. It has a lot of useful commands from moderation to information commands. With about 80 commands, it can do many features such as moderation on the server, informing everything that is happening on the server, beautifying the profile picture with different effects, playing games with a bot. With the "help" command, you can access all its features and use it to make your server a better place.
