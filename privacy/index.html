PRIVACY POLICY

User Data Handling and Usage:

For each direct interaction with the bot, I collect and store the name and ID of the user and Discord server.

The way I use this data is for statistical, analytical, and quality purposes, as well as monitoring for any bad content that breaks the Terms of Service, such as harmful usernames or server names. I store basic user and server data for the purposes of being able to block users and servers from using the bot if they are found to be abusing the bot.

Where Do We Store This Data:

We store the data in a secured MongoDB database hosted on our servers. Only the Berkobot's owners are have access to the database.

Deleting Your Data:

Commands that using data has "reset" option so users can use it to delete their own data. (example : /modlog-reset)
